---
title: Model Not Responding
description: What to do when your AI isn't replying.
---

## Try refreshing the page

Sometimes the simplest fix. Refresh and try your message again.

## Switch to a different model

Use the model picker dropdown at the top of the chat to select a different model. If the current model has an outage or issue, another one will work.

## BYOK users — check your API key

If you're on the BYOK plan:

1. Make sure your API key is entered correctly in Settings → API Keys → Environment Variables
2. Check that your account with the provider (OpenAI, Anthropic, OpenRouter) has credits/quota available
3. Try switching to a different model from your provider

## Check for service outages

If nothing is working, the underlying AI provider may have an outage. Check:
- [status.openai.com](https://status.openai.com)
- [status.anthropic.com](https://status.anthropic.com)
- [openrouter.ai](https://openrouter.ai) — check their Discord or status page

## Still not working?

Post in [Discord](https://discord.gg/uEugVKA8PH) with your subdomain name and what you're seeing. We'll check the logs and get you sorted.
