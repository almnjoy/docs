---
title: Adding Your API Key
description: Step-by-step guide to adding your own API key to your BYOK instance.
---

Once you're logged into your AI, follow these steps to add your API key.

<Steps>
  <Step title="Open Settings">
    In the chat interface, look for the **Settings** icon (usually a gear icon in the sidebar or top menu). Click it.
  </Step>

  <Step title="Go to API Keys">
    Inside Settings, find the **API Keys** section and click **Add**.
  </Step>

  <Step title="Add an Environment Variable">
    You'll see an option for **Environment Variable Overrides**. Add one of the following depending on your provider:

    | Provider | Variable name | Where to get the key |
    |---|---|---|
    | OpenRouter | `OPENROUTER_API_KEY` | openrouter.ai → Keys |
    | OpenAI | `OPENAI_API_KEY` | platform.openai.com → API Keys |
    | Anthropic | `ANTHROPIC_API_KEY` | console.anthropic.com → API Keys |

    Paste your key as the value and save.
  </Step>

  <Step title="Pick your model">
    Go back to the chat and use the model picker dropdown to select a model from your provider. Your key will be used automatically.
  </Step>
</Steps>

<Note>
**OpenRouter is recommended for BYOK users.** It gives you access to models from OpenAI, Anthropic, Google, and more — all with one key. Sign up free at openrouter.ai.
</Note>

## Troubleshooting

**Model shows an error after adding the key**
- Double-check the variable name is exact (case-sensitive)
- Make sure the key is valid and has credits/quota available
- Try a different model from the picker

**Not sure which model to pick**
- OpenRouter: `openai/gpt-4.1-mini` is a solid, cheap starting point
- OpenAI: `gpt-4o-mini` for fast and affordable
- Anthropic: `claude-3-5-haiku-20241022` for a good balance
