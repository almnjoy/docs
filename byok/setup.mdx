BYOK (Bring Your Own Key) lets you connect your existing OpenAI or Anthropic API key to your MadeForMeAI instance. You pay the AI provider directly for token usage — we charge $15/month to run your private setup.

## Before you start

You'll need an active API account with one of the following:
- [OpenAI](https://platform.openai.com) — create an API key under API Keys
- [Anthropic](https://console.anthropic.com) — create an API key under API Keys
- [OpenRouter](https://openrouter.ai) — create an API key in your dashboard (gives access to 300+ models)

## Adding your API key

Your Setup Assistant will walk you through this on first login. You can also do it manually:

1. Go to your control panel at `yourname-soul.madeformeai.com`
2. Navigate to **Settings → API Keys**
3. Paste your API key and save

Your key is stored encrypted. We never log or expose it.

## Choosing your models

Once your key is connected, you can select which models to use in your AI's settings. The models available depend on which provider's key you've added.

<Note>OpenRouter is the most flexible option — one key gives you access to OpenAI, Anthropic, Google, and hundreds of other models. Recommended if you want options.</Note>

## Usage and costs

You pay your AI provider directly for every token used. Your MadeForMeAI subscription ($15/mo) covers the hosting, memory, workspace, and all platform features — not the AI tokens.

Monitor your usage directly in your provider's dashboard:
- OpenAI: platform.openai.com/usage
- Anthropic: console.anthropic.com/usage
- OpenRouter: openrouter.ai/activity

## Switching providers

You can update your API key at any time from the control panel. Switching from OpenAI to Anthropic (or vice versa) just requires pasting a new key.
